Detailed Description

Struct to create only one type to handle the interactions with a card in the game. The struct contains three variables: side, if 1 then insert the card on the left side otherwise on the right; swap, if 1 then the values of the card are swapped otherwise nothing happens; confirm, if 1 then the choice is confirmed.
